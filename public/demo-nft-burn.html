♻ + 🔥 NFT Burn Feature

Brainstorm: mechanics, UX flows, and visual directions

Approach — how burning works
⭐ Recommended

A — Direct Burn + SMELT reward

User selects worthless/spam NFTs. Each NFT is burned on-chain (destroying it permanently). User gets back the ~0.002 SOL rent + bonus SMELT reward proportional to how many they burn.

✓ Clean, irreversible — feels meaningful

✓ Fits existing SMELT reward loop perfectly

✓ Reclaims rent + cleans wallet state

✗ Permanent — users may hesitate on NFTs they're unsure about

B — Burn-to-earn with tiered rewards

Same burn mechanic, but rewards scale by NFT "rarity" or collection floor. Verified junk = base reward. Semi-known collections = 2× SMELT. User sees preview of reward before confirming.

✓ More exciting — gamified

✓ Drives higher engagement

✗ Needs reliable floor/rarity data source (DAS API)

✗ More complex to implement fairly

C — Burn + Mint receipt NFT

User burns N spam NFTs, receives a "Recycler" soulbound badge NFT as proof. Badge shows how many they've burned total. Gamified identity — collector proof of cleanup.

✓ Novel — creates a collectible incentive

✓ Viral / shareable identity

✗ Requires minting infrastructure

✗ Counter-intuitive (burning creates NFT)

How spam NFTs get flagged
🔴
Auto-flagged SPAM
Zero verified collection authority · 0 floor price · airdropped to wallet (not bought) · unverified creators · name contains "free", "claim", "visit"
🟡
Unknown — user decides
No floor data available but has metadata · not auto-selected · user can manually check/burn
🔒
Protected — never shown
Verified collection with floor ≥ 0.1◎ · never shown as burnable even if user requests scan
📡
Data source
Helius DAS API already in use — getAssetBatch returns collection, verified, floor, creators. No new API needed.
Where it lives in the app
⭐ Option 1 — Tab on home page
Add "NFTs" tab next to "Tokens" on the recycle page. Same flow, same screen, same SMELT reward loop. Zero navigation overhead.
Option 2 — Dedicated /burn page
Separate page in nav: "Burn NFTs". Richer UI, fire animations, more dramatic. Separate brand feel from the token recycler.
Option 3 — Part of recycle flow
After scanning tokens, show "Also found 3 spam NFTs" card at the top. One extra step in the same flow. Feels natural but mixed concerns.
